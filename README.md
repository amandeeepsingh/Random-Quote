# Random-Quote
A Random Quote App built using React JS

> Demo: https://amandeeepsingh.github.io/random-quote/

# Installing and runing:

* ` git clone https://github.com/amandeeepsingh/Random-Quote.git `
* ` npm install `
* ` npm start `
